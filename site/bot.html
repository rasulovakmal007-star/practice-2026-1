Интерактивный Telegram-бот

В рамках вариативной части проекта (Практическая реализация технологии) нашей командой был спроектирован и разработан Telegram-бот «ЭкоМедиа». Он служит интерактивным продолжением нашего медиа-ресурса, позволяя пользователям получать знания об экологии человека и проверять свои привычки прямо в мессенджере.

Запустить бот в Telegram

Ключевые возможности бота

📝 Эко-викторина

Интерактивный тест из 3-х комплексных вопросов о влиянии микропластика, загрязнении питьевой воды и гаджетах. Бот отправляет развернутые научные пояснения после каждого ответа.

💡 Эко-советы

Выдача случайных практических рекомендаций по улучшению повседневной экологии (правила фильтрации воды, отказ от пластика, проветривание).

💤 Тест цифровой гигиены

Диагностический опросник с подсчетом баллов. Позволяет пользователю оценить, насколько здоровы его цифровые привычки перед сном, и получить персональные советы.

🌐 Интеграция с сайтом

Возможность быстро перейти на основной сайт проекта для подробного изучения статей и отчетов.

Архитектура и стек технологий

Бот написан на языке Python с использованием библиотеки pyTelegramBotAPI. Логика основана на событийной модели (Message handlers) и обработке обратных вызовов (Callback queries) для интерактивных инлайн-кнопок.

Фрагмент исходного кода

Пример реализации обработчика запуска бота и главного меню в файле src/main.py:

import telebot
from telebot import types

TOKEN = "8789087980:AAFaJprDF1jCo3u8mgBchWeef7i_qTnTV_8"
bot = telebot.TeleBot(TOKEN)

# Функция создания главного меню
def get_main_keyboard():
    markup = types.ReplyKeyboardMarkup(resize_keyboard=True, row_width=2)
    btn_quiz = types.KeyboardButton("📝 Пройти эко-тест")
    btn_tips = types.KeyboardButton("💡 Получить эко-совет")
    btn_hygiene = types.KeyboardButton("💤 Тест цифровой гигиены")
    btn_site = types.KeyboardButton("🌐 О проекте и сайт")
    markup.add(btn_quiz, btn_tips, btn_hygiene, btn_site)
    return markup

# Обработчик команды /start
@bot.message_handler(commands=['start'])
def send_welcome(message):
    bot.send_message(
        message.chat.id, 
        "👋 Приветствуем в ЭкоМедиа Боте! Используйте меню ниже:", 
        reply_markup=get_main_keyboard()
    )
