# Документация по практике

## 1. Введение
В рамках проектной (учебной) практики был реализован проект «ЭкоМедиа», направленный на популяризацию здорового образа жизни и экологического мышления в современной городской среде. В ходе практики команда из двух студентов выполнила как базовую часть задания (настройка репозитория, создание статического сайта), так и вариативную часть (разработка Telegram-бота).

## 3. Базовая часть задания

### 3.1. Настройка репозитория
Был создан и настроен групповой репозиторий на базе предоставленного шаблона `mospol/practice-2025-1`.
Освоены базовые команды работы с Git: клонирование, создание веток, фиксация изменений (`git commit`) и их отправка (`git push`).

### 3.2. Создание веб-сайта
Разработан статический веб-сайт с использованием HTML и CSS. Сайт расположен в папке `site/` и включает в себя:
- **Главную страницу** (`index.html`) с аннотацией проекта.
- **Страницу «О проекте»** (`about.html`), где описаны актуальность, цели и задачи.
- **Страницу «Участники»** (`participants.html`) с распределением обязанностей.
- **«Журнал прогресса»** (`journal.html`), где задокументированы ключевые этапы (сбор материалов, создание контента, анализ аудитории).
- **Раздел «Ресурсы»** (`resources.html`) с полезными ссылками на внешние материалы.

Оформление выполнено в современном стиле (тёмная тема, эффекты glassmorphism) с использованием полностью уникальных графических решений.

## 4. Вариативная часть: Практическая реализация технологии

В качестве вариативного задания (Вариант 2) был разработан интерактивный **Telegram-бот «ЭкоМедиа»** (`@ecomediaprobot`).

### 4.1. Технологический стек
- Язык программирования: **Python 3**
- Библиотека: `pyTelegramBotAPI`
- Хранение состояний: оперативная память (словари Python)

### 4.2. Функционал бота
- **Эко-викторина:** Интерактивный тест на знания о микропластике и качестве воды.
- **Тест цифровой гигиены:** Оценка привычек пользователя (время за экраном, фильтры синего света) с расчетом баллов и выдачей рекомендаций.
- **Эко-советы:** Генерация случайных полезных советов по улучшению качества повседневной жизни.
- **Интеграция с веб-ресурсом:** Прямые ссылки на сайт проекта.

Подробное техническое руководство с UML-диаграммами (Sequence, State и Flowchart) и инструкциями по развертыванию находится в файле [telegram_bot_guide.md](telegram_bot_guide.md).
